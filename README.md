# Data Structures and Algorithms in C
For Fun and Profit
## Build & Run
### Build the executable
```
cd <repo_directory/data_structure>
./build.sh
```
### Run the executable
```
cd <repo_directory/data_structure>
./main
```
### Clean build files
```
cd <repo_directory/data_structure>
./build.sh clean
```
